## Run Locally

Clone the project

```bash
git clone https://link-to-project
cd quotes-api
```

Create a virtual env and install dependencies

```bash
python3 -m venv venv
pip install -r requirements.txt
```

```bash
cp .env.example .env
```

Start the server

```bash
python3 app/main.py
```

Go to docs http://0.0.0.0:8000/docs
